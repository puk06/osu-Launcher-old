# osu! Launcher
プライベートサーバーなどに接続しやすくしてくれるソフトです。

# 使い方
まず、osu!を起動して自動検出ボタンを押して保存するか、手動入力でosu!フォルダの場所を指定してください。

次に接続したいサーバーを選ぶか、入力して下のosu!のボタンを押せば起動します。
